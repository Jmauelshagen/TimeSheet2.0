As an HR employee, you are assigned the task of maintaining employees' weekly timesheets.



General Info

After a successful login as an HR employee, you will reach a paysheet summaries page. Once on the paysheet summaries page, an HR employee can select a week ending date via a drop down tab in the middle of the page Once a week ending date is selected, employees with timesheets for that date will be listed.

Example

Exporting To Excel

The option to export a report to excel is also present near the bottom of the page, clicking it will allow the HR employee to choose a save location for the excel file.

Example

Opening & Editing Timesheets

Selecting an employee's name and ID will open an overview page that contains the employee's timesheet. While viewing the overview page for an employee, you will have the ability to edit the timesheet if there are errors. Saving an employee's timesheet information after making changes will send a generated email to that employee. If any fields are left blank or have errors, an indicator will pop up to show where changes need to be made.

Emailing Employees

There are two ways to email an employee: one method of emailing an employee is by the email section at the bottom of the page, the other method is via an email link where the employee's email is displayed. Upon clicking on an employee's email on the overview page, an email window will be displayed. Your HR email must be signed in and you will be able to compose an email to send to the selected employee.

Printing Timesheets

An HR employee may be able to print an employee's timesheet via the print button in the overview page. A print window will then pop up prompting the user to print to the default printer.
